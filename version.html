VERSION_MAJOR¶

Address: 0xe0007000 + 0x0 = 0xe0007000

Major git tag version. For example, this firmware was built from git tag v1.8.7, so this value is 1.

VERSION_MINOR¶

Address: 0xe0007000 + 0x4 = 0xe0007004

Minor git tag version. For example, this firmware was built from git tag v1.8.7, so this value is 8.

VERSION_REVISION¶

Address: 0xe0007000 + 0x8 = 0xe0007008

Revision git tag version. For example, this firmware was built from git tag v1.8.7, so this value is 7.

VERSION_GITREV3¶

Address: 0xe0007000 + 0xc = 0xe000700c

Bits 24-31 of VERSION_GITREV. First 32-bits of the git revision. This documentation was built from git rev 0e5e760e, so this value is 241071630, which should be enough to check out the exact git version used to build this firmware.

VERSION_GITEXTRA1¶

Address: 0xe0007000 + 0x1c = 0xe000701c

Bits 8-9 of VERSION_GITEXTRA. The number of additional commits beyond the git tag. For example, if this value is 1, then the repository this was built from has one additional commit beyond the tag indicated in MAJOR, MINOR, and REVISION.
